## **1\. Meaning**

A **Random Variable** is a function or rule that assigns a **numerical value** to every outcome of a random experiment.

## **6\. Characteristics**

* A random variable takes **different possible values** depending on the random outcome.

* It has an associated **probability distribution**, which shows how likely each value is.

* The total probability for all possible values is always **1**.

* It can be either **discrete** (countable values) or **continuous** (measurable values).
