# **CRUD Operations**

CRUD refers to the four fundamental operations performed on a database:

1. **Create**

   * Adds new data

   * SQL: `INSERT`

   * MongoDB: `insertOne()`

2. **Read**

   * Retrieves existing data

   * SQL: `SELECT`

   * MongoDB: `find()`

3. **Update**

   * Modifies existing records

   * SQL: `UPDATE`

   * MongoDB: `updateOne()`

4. **Delete**

   * Removes records

   * SQL: `DELETE`

   * MongoDB: `deleteOne()`
