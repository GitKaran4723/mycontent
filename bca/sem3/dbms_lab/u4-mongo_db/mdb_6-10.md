# **MONGO DB (6-10 Queries)**

### **6\. Retrieve all documents where part\_name is 'P1' OR price is less than 200**  
   
**Query:**  
>db.suppliers.find  (  {  $or:\[  {pname:"p1"},{price:{$lt:200}}  \]  },  {\_id:0}  )

### **7\. Update the record of 'Geeta', set city='Bombay' and phoneno='11223344'**

**Query:**  
>db.suppliers.updateOne({name:"Geeta"},{$set:{city:"Bombay",phone:11223344}})

db.suppliers.find({ name: "Geeta" },{\_id:0})

**Output:**  
{  
  name: 'Geeta',  
  city: 'Bombay',  
  color: 'red',
