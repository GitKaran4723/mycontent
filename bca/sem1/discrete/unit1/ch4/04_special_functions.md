# Special Functions 

In discrete structures, **special functions** are functions with **simple and fixed behavior** that are frequently used in set theory and logic.

---

# 1. Constant Function

**Definition:**
A function that assigns the **same value to every element** of the domain.

$$
f(x) = c
$$

**Example:**
If $f: A \to B$, then
$$
f(x) = 3 \quad \forall x \in A
$$

---

# 2. Identity Function

**Definition:**
A function that maps **each element to itself**.

$$
f(x) = x
$$

**Example:**
For set $A = {1,2,3}$,
$f(2) = 2$

---

# 3. Zero Function

**Definition:**
A special constant function whose value is **always zero**.

$$
f(x) = 0
$$

---

# 4. Successor Function

**Definition:**
Maps each natural number to its **next number**.

$$
f(x) = x + 1
$$

**Example:**
$f(5) = 6$

---

# 5. Predecessor Function

**Definition:**
Maps a number to its **previous number** (where defined).

$$
f(x) = x - 1
$$

---

# 6. Projection Function

**Definition:**
A function that returns **one component of an ordered pair or tuple**.

$$
f(x,y) = x \quad \text{or} \quad f(x,y) = y
$$

---

# 7. Characteristic (Indicator) Function

**Definition:**
Indicates whether an element belongs to a set.

$$
\chi_A(x) =
\begin{cases}
1, & x \in A \
0, & x \notin A
\end{cases}
$$
