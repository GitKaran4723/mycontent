# **8085 Instruction Set**

---

## **1\. Introduction**

The **8085 microprocessor** has a **well-defined set of instructions** that allow it to perform arithmetic, logic, data transfer, program control, and machine operations.

* **Total instructions:** 74

* **Total opcodes:** 246

* **Instruction size:** 1, 2, or 3 bytes

**Purpose of the instruction set:**

To provide the microprocessor with commands for manipulating data, transferring information, performing arithmetic and logic operations, and controlling program flow.

---

## **2\. Classification of Instructions**

The 8085 instructions are classified based on their **functionality**.

---

### **2.1 Data Transfer Instructions**

* **Purpose:** Move data between **registers, memory, and I/O ports**.

* **Flags:** Usually not affected.

**Common Instructions:**

| Instruction | Description | Example |
| ----- | ----- | ----- |
| `MOV Rd, Rs` | Copy data from source register to destination | `MOV A, B` |
| `MVI R, data` | Load immediate data into register | `MVI A, 55H` |
| `LDA addr` | Load accumulator from memory | `LDA 2050H` |
| `STA addr` | Store accumulator into memory | `STA 2052H` |
| `LHLD addr` | Load H-L pair directly from memory | `LHLD 3000H` |
| `SHLD addr` | Store H-L pair into memory | `SHLD 3000H` |
| `LXI RP, data16` | Load 16-bit data into register pair | `LXI H, 3000H` |
| `IN port` | Read data from I/O port | `IN 01H` |
| `OUT port` | Send data to I/O port | `OUT 02H` |
| `XCHG` | Exchange D-E and H-L register pair | `XCHG` |

---

### **2.2 Arithmetic Instructions**

* **Purpose:** Perform arithmetic operations like **addition, subtraction, increment, decrement**.

* **Flags affected:** **Z, S, P, CY, AC**

**Common Instructions:**

| Instruction | Description | Example |
| ----- | ----- | ----- |
| `ADD R/M` | Add register/memory to accumulator | `ADD B` |
| `ADC R/M` | Add with carry | `ADC C` |
| `SUB R/M` | Subtract register/memory from accumulator | `SUB D` |
| `SBB R/M` | Subtract with borrow | `SBB E` |
| `INR R/M` | Increment register/memory by 1 | `INR A` |
| `DCR R/M` | Decrement register/memory by 1 | `DCR B` |
| `INX RP` | Increment register pair | `INX H` |
| `DCX RP` | Decrement register pair | `DCX D` |
| `DAD RP` | Add register pair to H-L pair | `DAD B` |
| `DAA` | Decimal adjust accumulator (BCD operations) | `DAA` |

---

### **2.3 Logical Instructions**

* **Purpose:** Perform **bitwise logical operations** on accumulator.

* **Flags affected:** **Z, S, P, CY (sometimes reset), AC**

**Common Instructions:**

| Instruction | Description | Example |
| ----- | ----- | ----- |
| `ANA R/M` | AND accumulator with register/memory | `ANA B` |
| `ORA R/M` | OR accumulator with register/memory | `ORA C` |
| `XRA R/M` | XOR accumulator with register/memory | `XRA D` |
| `CMA` | Complement accumulator | `CMA` |
| `CMP R/M` | Compare accumulator with register/memory | `CMP E` |
| `RLC` | Rotate accumulator left | `RLC` |
| `RRC` | Rotate accumulator right | `RRC` |
| `RAL` | Rotate accumulator left through carry | `RAL` |
| `RAR` | Rotate accumulator right through carry | `RAR` |
| `CPI data` | Compare accumulator with immediate data | `CPI 55H` |

---

### **2.4 Branch / Program Control Instructions**

* **Purpose:** Control **program flow** – jumps, calls, returns, and conditional execution.

* **Flags affected:** **Program Counter changes; some instructions check flags.**

**Common Instructions:**

| Instruction | Description | Example |
| ----- | ----- | ----- |
| `JMP addr` | Unconditional jump | `JMP 2050H` |
| `JC addr` | Jump if carry set | `JC 3000H` |
| `JNC addr` | Jump if carry not set | `JNC 3000H` |
| `JZ addr` | Jump if zero | `JZ 2050H` |
| `JNZ addr` | Jump if not zero | `JNZ 2050H` |
| `CALL addr` | Call subroutine | `CALL 3000H` |
| `RET` | Return from subroutine | `RET` |
| `RST n` | Restart (call fixed address) | `RST 7` |

---

### **2.5 Stack / Machine Control Instructions**

* **Purpose:** Control **CPU operations and stack operations**.

* **Flags affected:** Usually none.

**Common Instructions:**

| Instruction | Description | Example |
| ----- | ----- | ----- |
| `PUSH RP` | Store register pair on stack | `PUSH B` |
| `POP RP` | Retrieve register pair from stack | `POP D` |
| `XTHL` | Exchange top of stack with H-L pair | `XTHL` |
| `SPHL` | Load stack pointer from H-L pair | `SPHL` |
| `NOP` | No operation | `NOP` |
| `HLT` | Halt CPU | `HLT` |
| `DI` | Disable interrupts | `DI` |
| `EI` | Enable interrupts | `EI` |

---

## **3\. Instruction Size**

8085 instructions are **1, 2, or 3 bytes** depending on whether they have:

* **No operand** → 1 byte (e.g., `CMA`)

* **8-bit operand** → 2 bytes (e.g., `MVI A, 55H`)

* **16-bit operand (address)** → 3 bytes (e.g., `LDA 2050H`)

---

## **4\. Flags Affected by Instructions**

| Instruction Type | Flags Affected |
| ----- | ----- |
| Data Transfer | Usually none |
| Arithmetic | Z, S, P, CY, AC |
| Logical | Z, S, P, CY (C sometimes reset), AC |
| Branch/Control | PC changes; may check flags |
| Stack/Control | Usually none |

---

## **5\. Summary Table of Instruction Categories**

| Category | Purpose | Examples |
| ----- | ----- | ----- |
| Data Transfer | Move data between registers, memory, I/O | MOV, MVI, LDA, STA, LXI, LHLD, SHLD, IN, OUT, XCHG |
| Arithmetic | Add, subtract, increment, decrement | ADD, ADC, SUB, SBB, INR, DCR, INX, DCX, DAD, DAA |
| Logical | AND, OR, XOR, complement, rotate, compare | ANA, ORA, XRA, CMA, CMP, RLC, RRC, RAL, RAR, CPI |
| Branch / Program Control | Jump, call, return, conditional execution | JMP, JC, JNC, JZ, JNZ, CALL, RET, RST |
| Stack / Machine Control | Push/pop stack, enable/disable interrupts, halt | PUSH, POP, XTHL, SPHL, NOP, HLT, DI, EI |

---

## **6\. Key Points**

* 8085 is an **8-bit microprocessor**, but **addresses are 16-bit**.

* Instruction set covers **all operations needed for typical programs**.

* Instructions can be **immediate, register, memory, or I/O based**.

* **Efficient programming** requires proper use of instruction types and minimal memory usage.
